1. Introduction

Institution Office Task Reminder is a professional desktop-based task management and reminder system designed for offices, schools, colleges, and institutions.

Management Scope

Deadlines: Keep track of critical completion dates.
Follow-up: Track activity progression.
Daily Reminders: Automated notification cycle.
Overdue tracking: Visual indicators for missed tasks.
Staff coordination: Multi-user shared environment.
Workflows: Systematic task lifecycle management.

Important Notes

  • Telegram integration is outgoing-only
  • No incoming command polling (/pending, /status)
  • System sends automated reminders and summaries
  • Multi-user synchronization is fully supported

8.2 Contact Mapping & Setup

Use **Manage Contacts** in Cloud settings to link staff names with their unique Telegram Chat IDs.

How to create a Bot
  1. Search for @BotFather on Telegram.
  2. Send /newbot and follow instructions.
  3. Copy the API Token provided at the end.
  4. Paste this Token into the app's Cloud settings.
How to find Chat ID
  1. Search for @GetMyID_Bot on Telegram.
  2. Start the bot to see your numeric Chat ID.
  3. Add this ID to Manage Contacts for each user.

Automation features require a valid signed license and enabled device activation.

11. Day-to-Day Operation

11.1 Synchronization

Configure intervals (10s to 30 mins) in Settings → General for background updates. Manual sync available via "Sync Now".

11.2 Conflict Resolution

System detects multi-user collisions. Options: "Keep Offline Edit", "Overwrite with Server", or "Restore As New Task".

11.3 File Attachments

Use the "Browse..." button in the Add Task dialog to select local files. The software automatically uploads them to Google Drive and saves the cloud link.

11.4 Bangla Unicode Support

Native support for Bijoy and Avro Unicode typing is enabled by default. No special configuration is required for typing in Bengali.

11.6 Data Export

Use the "Export" button in the toolbar to save the current table view as a CSV file. Filter by date range or responsible person before exporting.

11.7 Branding & Identity

Upload your **Institution Logo (PNG)** and set a custom name in Settings → Branding. You can choose to apply this branding to the application header or the window title.

11.8 Administrative Controls

Admins can update global configurations such as **Categories**, **Importance levels**, **Current Session**, and the **Default Responsible person** via the Institution tab.

11.9 General Settings

Manage startup & branding

Configure Auto-Sync, **Start on Boot**, and **Start Minimized to Tray** directly from the UI for a seamless background experience.

11.10 User Display Name

Set a unique **Display Name** in General settings. This helps track which specific user or department added or updated a task in the audit logs.

13. Troubleshooting

Connection Test Fails

Verify the Apps Script deployment URL and Security Token. Ensure "Access: Anyone" was selected during deployment.

Telegram Message Not Sent

Ensure the Chat ID is mapped correctly in Staff Contacts and the Bot Token is pushed to Script Properties.

Sync Conflict Detected

This occurs when two users edit the same task simultaneously. Choose "Overwrite with Server" to discard local changes, or "Restore As New" to keep both versions.

Action Requires Device Activation

This means your 15-day trial has expired or you are on a new device. Contact support to obtain a signed license file for full access.

File Upload Timeout

Large files may timeout on slow networks. The system now uses chunked uploads, but ensure your internet connection is stable during the process.

14. Best Practices

  • Use Setup Wizard for all deployments.
  • Restrict Admin access to one computer.
  • Enable automatic synchronization (5-15 mins).
  • Keep Bot Tokens restricted to Script Properties.
  • Use consistent task Statuses across teams.
  • Protect the original signed license files.
