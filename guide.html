Getting Started

Quick Start Guide

Follow these steps to start using the software immediately after installation.

Install the software — Download the installer from the Download page and run it on your Windows computer.

Trial starts automatically — On first launch, the 15-day free trial begins without any manual activation. You have full access immediately.

Add your first task — Use the "Add Task" option. Enter the task title, category, deadline, priority, and any relevant notes.

Set categories and deadlines — Assign each task to a category (Academic, Financial, Exam, etc.) and set a clear deadline for tracking.

Configure user roles — Set up Admin, Editor, and Viewer accounts according to your institutional staff structure.

Use the dashboard for monitoring — The dashboard gives you an at-a-glance view of active, overdue, and today's tasks every time you open the software.

Import your license when ready — After purchasing a commercial license, go to About & License section and import the signed license file to activate full access.

Reference

Common Tasks at a Glance

➕

Adding a Task

Click "Add Task," fill in the title, select a category, set a deadline and priority, and save. The task appears immediately in your list and dashboard.

📊

Reading the Dashboard

The dashboard shows total active tasks, overdue count, today's tasks, and category distribution. Check it daily to stay on top of institutional responsibilities.

🔍

Searching & Filtering

Use the search bar to find tasks by keyword. Use filters to narrow by category, status (active/overdue/completed), or date range.

🛡️

Importing a License

Navigate to About & License from the menu. Click "Import License," select your signed license file, and confirm. Full access is activated immediately.
